gifCatch options

gif.js is used to generate the gifs!

▷ show information

This value corresponds to the number of web workers to spawn, which work in the background to create the gif.

Increasing the number can help speed up the process, but too many workers may result in poor performance.

▷ show information

This value corresponds to the pixel sample interval.
The lower the value, the higher the quality.

▷ show information

This value indicates the number of frames to collect.

▷ show information

This value indicates the time in milliseconds between frames when recording .

▷ show information

This value indicates the time in milliseconds between frames for the resulting gif .
